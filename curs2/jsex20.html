Dacă cuvântul cheie var sau let este utilizat la definirea variabilei în cadrul unei funcții sau al unui alt domeniu non-global, atunci acea variabilă nu este globală.
Dacă cuvântul cheie var sau let nu este utilizat la definirea variabilei înainte de un nume de variabilă, atunci acea variabilă este globală.
var sau let este obligatoriu dacă nu se inițializează variabila
Se recomanda declararea variabilelor cu let https://www.w3schools.com/js/js_let.asp
